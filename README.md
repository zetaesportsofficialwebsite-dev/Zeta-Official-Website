# ZeTa Esport

## Overview

Esport club website template for ZeTa Esport. Showcases team, roster, match history, and news. Built as a plain HTML/CSS/JS static site — no build step required.

## Getting Started

```bash
# No dependencies to install
# Open index.html in a browser, or serve locally:
npx serve .
```

## Structure

```
index.html        # Homepage
about.html         # About / founder's story
committee.html      # Real committee org chart + hierarchy
events.html         # Real tournament/event history
partners.html        # Real partners & sponsors
css/              # Stylesheets
js/               # Scripts
assets/           # Images and media (org chart photos)
```
